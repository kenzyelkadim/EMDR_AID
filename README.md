# EMDR_AID
EMDR Therapy Aid
A browser-based tool designed to assist therapists during EMDR sessions. Displays a bilateral stimulus dot that moves across the screen at a controlled pace, reducing the need for manual guidance and letting the therapist focus on the client.
Features

Adjustable speed and pass count via sliders
Automatic stop when the set number of passes is reached
Pass counter displayed in real time
Clean, distraction-free dark interface

Tech Stack
JavaScript · HTML · CSS
